# Employee-tracker-xc

## Description
This project tracks employees for a company, and allows you to also update employees by role, department. You can also add employees, departments, or roles.

## Built With
* Javascript
* Node
* Express.js
* Inquirer
* MySQL2
* Sequelize
* Figlet
* Ctable
